# Integrated Urban Flood Prediction System 🌊

## Overview
This project predicts urban flood risks using rainfall and water level data and visualizes risk zones on a map.

## Features
- Flood risk prediction (High, Moderate, Low)
- Heatmap visualization using Leaflet
- REST API using Flask

## How to Run
1. Go to backend folder
2. Install requirements: pip install -r requirements.txt
3. Run server: python app.py
4. Open frontend/index.html in browser
